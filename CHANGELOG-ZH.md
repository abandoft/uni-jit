## 0.1.2

- 新增类型化 Float64 SSA 常量与算术、IEEE-754 解释器 oracle、带优化的
  溢出处理，以及三个目标架构上的原生编码。
- 为全部原生后端新增 CFG 块内寄存器分配和可安全处理环路的并行寄存器复制，
  消除热循环中的栈往返。
- 将项目自有的前端、工具和文档目录统一改为单数形式：`frontend/`、
  `tool/` 与 `doc/`。
- 新增一个结构化 Lua 5.5 数值 `for` 循环的受守卫编译，覆盖 CFG 循环携带值、
  零次迭代语义与三引擎测试。
- 新增 AArch64、x86-64、RISC-V 64 上带范围校验的 CFG 原生降级，并以
  并行块参数复制正确处理循环回边。
- 新增使用显式块参数的 CFG SSA、支配关系验证，以及支持分支和循环并带执行
  预算的参考解释器。
- 新增 Lua 三引擎整数基准与锁定 ABI 的调用快路径；在本次调用边界工作负载上
  性能超过 stock Lua。
- 新增受守卫保护的 stock Lua 5.5 字节码到原生代码前端，首批支持直线
  整数函数，并配套差分与生命周期测试。
- 新增用于语言基准的隔离式 Lua 5.5 与 LuaJIT 参考构建。

## 0.1.1

- 新增经过验证的直线整数 SSA 表示与参考解释器。
- 新增自主实现的 AArch64、x86-64、RISC-V 64 原生代码生成器。
- 新增支持寄存器溢出的线性扫描分配器与 W^X 代码发布机制。
- 新增覆盖 macOS、Linux、Windows 的确定性差分测试。
- 新增可安装的 CMake 包元数据与跨平台 CI 矩阵。
- 新增规范化 SSA 优化流程与可复现核心基准。
- 新增十进制版本校验与由变更日志驱动的 GitHub Release 流程。
